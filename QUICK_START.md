## 🔧 Minimal Customization

### Change App Name
Find this in `index.html`:
```html
<a href="#" class="logo">AeroTrack</a>
```
Change "AeroTrack" to your airport name.

### Change Colors
Find this in `css/style.css`:
```css
:root {
  --primary-blue: #00d4ff;
```
Change hex colors to your brand colors.

### Change Dummy Data
Find this in `js/script.js`:
```javascript
const mockFlights = ['AI-202', 'AA-456', ...];
```
Update with your test data.

## 📞 Troubleshooting

**Styles not loading?**
- Check that `css/style.css` exists
- Try refreshing (Ctrl+Shift+R)

**JavaScript not working?**
- Check that `js/script.js` exists
- Open browser console (F12) for errors

**Buttons not responding?**
- Make sure JavaScript file loaded
- Try different browser
- Clear cache and reload

**Page won't open?**
- Check file path is correct
- Make sure all files are in place
- Try drag & drop method
